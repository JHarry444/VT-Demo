Sources

Use F12 or RMB > inspect and open the Sources tab.

In this tab we cann see all of the source files for this web page (typically HTML, CSS and JS).

Clicking the DEMO button will trigger a function in the JS - we can observe its effects in the Console tab.

In JS files it is possible to apply breakpoints to pause code mid-execution.

Let's apply a breakpoint and then rerun the previous function.

While the code is paused on a breakpoint we can interrogate the values of any variables in the Breakpoints subsection and we can even perform operations with them in the Console tab!
