# DevTools Debugging Course

Welcome to the DevTools Debugging Course! This course is designed to teach non-technical individuals how to effectively use the Developer Tools in Microsoft Edge to debug JavaScript applications. Throughout this course, you will engage in hands-on exercises that will familiarize you with various features of DevTools, including the Elements, Console, Sources, Network, and Application tabs.

## Course Objectives

- Understand the purpose and functionality of Developer Tools in web browsers.
- Learn how to inspect and modify HTML and CSS using the Elements tab.
- Utilize the Console tab to log messages and debug JavaScript code.
- Debug JavaScript applications using the Sources tab.
- Monitor network requests and analyze responses with the Network tab.
- Inspect web storage and service workers using the Application tab.

## Course Structure

The course consists of five exercises, each focusing on a specific aspect of DevTools:

1. **Exercise 1: Elements**
   - Focus: Inspecting and modifying HTML and CSS.
   - Files: `exercises/exercise-1-elements/index.html`, `exercises/exercise-1-elements/instructions.md`

2. **Exercise 2: Console**
   - Focus: Logging messages and debugging JavaScript.
   - Files: `exercises/exercise-2-console/index.html`, `exercises/exercise-2-console/instructions.md`

3. **Exercise 3: Sources**
   - Focus: Debugging JavaScript using breakpoints.
   - Files: `exercises/exercise-3-sources/index.html`, `exercises/exercise-3-sources/app.js`, `exercises/exercise-3-sources/instructions.md`

4. **Exercise 4: Network**
   - Focus: Monitoring network requests.
   - Files: `exercises/exercise-4-network/index.html`, `exercises/exercise-4-network/app.js`, `exercises/exercise-4-network/instructions.md`

5. **Exercise 5: Application**
   - Focus: Inspecting storage and service workers.
   - Files: `exercises/exercise-5-application/index.html`, `exercises/exercise-5-application/app.js`, `exercises/exercise-5-application/instructions.md`

## Setup Instructions

To get started with the exercises, follow these steps:

1. Clone the repository to your local machine.
2. Open the `index.html` file for the exercise you want to work on in Microsoft Edge.
3. Open Developer Tools by pressing `F12` or right-clicking on the page and selecting "Inspect".
4. Follow the instructions provided in the corresponding `instructions.md` file for each exercise.

## Solutions

Solutions for each exercise are available in the `solutions` directory. You can refer to these solutions after attempting the exercises to check your understanding and see the expected outcomes.

## Conclusion

By the end of this course, you will have gained practical experience using Developer Tools to debug JavaScript applications. We encourage you to explore beyond the exercises and continue practicing your skills in web development and debugging. Happy learning!
