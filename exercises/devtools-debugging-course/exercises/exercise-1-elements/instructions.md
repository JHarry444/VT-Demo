# Exercise 1: Using the Elements Tab in DevTools

## Objective
In this exercise, you will learn how to use the Elements tab in the DevTools of your browser to inspect and modify HTML and CSS on a simple webpage that utilizes Bootstrap components.

## Instructions

1. **Open the Webpage**:
   - Open the `index.html` file located in the `exercises/exercise-1-elements` directory in your browser.

2. **Access DevTools**:
   - Right-click anywhere on the webpage and select "Inspect" or press `F12` to open the DevTools.

3. **Explore the Elements Tab**:
   - Navigate to the "Elements" tab in DevTools. Here, you will see the HTML structure of the webpage.
   - Hover over different elements in the HTML to see their corresponding areas highlighted on the webpage.

4. **Modify HTML**:
   - Select an element (e.g., a heading or a button) in the Elements tab.
   - Right-click on the selected element and choose "Edit as HTML". 
   - Change the text or add new elements (like a new paragraph or image) and press `Enter` to see the changes reflected on the webpage.

5. **Inspect CSS Styles**:
   - With an element selected, look at the "Styles" pane on the right side of the Elements tab.
   - You can see the CSS rules applied to the selected element. Try modifying some of the CSS properties (like `color`, `font-size`, or `background-color`) to see how it affects the appearance of the element.

6. **Add New Styles**:
   - In the Styles pane, you can also add new CSS rules. Click on an empty space in the Styles pane and type a new CSS property (e.g., `border: 2px solid red;`) to apply it to the selected element.

7. **Take Notes**:
   - As you make changes, take notes on what you did and how it affected the webpage. This will help you understand the relationship between HTML and CSS.

8. **Reset Changes**:
   - Refresh the webpage to reset all changes made during this exercise.

## Conclusion
By the end of this exercise, you should be comfortable using the Elements tab to inspect and modify HTML and CSS. This skill is essential for debugging and enhancing web applications.
