# Exercise 2: Console Instructions

In this exercise, you will learn how to use the Console tab in the Developer Tools to debug JavaScript code. Follow the instructions below to complete the exercise.

## Objectives

- Understand how to access the Console tab in Developer Tools.
- Learn how to log messages to the console.
- Use the console to inspect variables and function outputs.
- Identify and fix errors in JavaScript code.

## Instructions

1. **Open the Webpage**: 
   - Open the `index.html` file located in the `exercise-2-console` folder in your web browser.

2. **Access Developer Tools**:
   - Right-click on the webpage and select "Inspect" or press `F12` to open Developer Tools.
   - Navigate to the **Console** tab.

3. **Logging Messages**:
   - In the `index.html` file, locate the JavaScript code section.
   - You will see a function that is supposed to log a message to the console. Ensure that the function is called correctly.
   - Modify the function to log a custom message of your choice.

4. **Inspecting Variables**:
   - Find a variable in the JavaScript code that holds a value.
   - Use `console.log()` to print the value of this variable to the console.
   - Refresh the webpage and check the console to see the output.

5. **Identifying Errors**:
   - Intentionally introduce a syntax error in the JavaScript code (e.g., remove a semicolon or misspell a variable name).
   - Observe how the console displays the error message.
   - Use the information provided in the console to fix the error.

6. **Debugging Functions**:
   - Locate a function that is not returning the expected result.
   - Use `console.log()` statements inside the function to track the flow of execution and the values of variables at different points.
   - Analyze the output in the console to identify where the logic may be failing.

7. **Final Review**:
   - Once you have completed the above tasks, review your changes.
   - Ensure that all console logs are meaningful and that any errors have been resolved.

## Conclusion

By the end of this exercise, you should be comfortable using the Console tab to log messages, inspect variables, and debug JavaScript code. Make sure to practice these skills as they are essential for effective web development.
